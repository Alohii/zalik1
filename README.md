# ACS Study Site

Interactive review and quiz page for Automatic Control Systems.

## What is included

- Review cards
- Practice questions
- Mixed quiz
- Fill-in-the-blank and calculation questions
- Handwriting and drawing canvas
- Model answers in English with Arabic explanations
- Homework sections HW1, HW2, and HW3

Open `index.html` directly in a browser, or publish this folder with GitHub Pages.
